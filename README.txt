SimpleID
========

SimpleID is a simple, personal OpenID provider written in PHP.

Latest Version
--------------

The latest version of SimpleID can be found in the SimpleID web page at
<http://simpleid.sourceforge.net/>

Installation
------------

For installation instructions, please see the SimpleID manual at
<http://simpleid.sourceforge.net/manual>.

Documentation
-------------

The SimpleID manual can be found at <http://simpleid.sourceforge.net/manual>.
